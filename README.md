# 2. 스터디 규칙
    
- 다음주 알고리즘 문제풀이 계획을 노션에 작성
- 유튜브 ‘이것이 코딩테스트다’ 수강
- 강의 내의 문제풀이 포함해서 최소 2개의 알고리즘 문제 풀고 정리
- 세션 전에 노션에 증빙자료 업로드
- 세션 중에 화면공유로 증빙자료 발표
- 다음주 알고리즘 문제풀이 계획을 노션에 작성
